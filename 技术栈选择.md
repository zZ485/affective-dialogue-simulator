## 技术栈推荐

### 前端
- **React 18** + **TypeScript**
- **状态管理**：Zustand（轻量，适合中小型项目）
- **UI 组件库**：Tailwind CSS + shadcn/ui（现代、可定制）
- **路由**：React Router v6
- **HTTP 客户端**：Axios

### 后端
- **框架**：FastAPI（Python，异步高性能，API 文档自动生成）
- **会话管理**：Redis（存储对话历史和内部状态）
- **数据库**：PostgreSQL（持久化用户数据、情境配置）
- **ORM**：SQLAlchemy 2.0
- **API 文档**：FastAPI 自带的 Swagger UI

### 大模型接入
- **HTTP 客户端**：httpx（异步）
- **Prompt 管理**：LangChain 或自建 Prompt Template 系统
- **模型调用**：aihubmix API
  - Model A：使用情感理解能力强的模型（如 Claude、GPT-4）
  - Model B：使用分析总结能力强的模型

### 基础设施
- **容器化**：Docker + Docker Compose
- **环境管理**：Poetry（Python 依赖）+ pnpm（前端依赖）
- **配置管理**：Pydantic Settings
- **日志**：Loguru

### 开发工具
- **代码规范**：ESLint + Prettier（前端）、Black + Ruff（后端）
- **类型检查**：TypeScript + mypy
- **版本控制**：Git

### 部署（可选）
- **前端托管**：Vercel / Netlify
- **后端部署**：Railway / Fly.io / 自建服务器
- **反向代理**：Nginx

这个技术栈的优势：
1. **快速开发**：FastAPI + React 都有成熟生态
2. **类型安全**：全栈 TypeScript/Python 类型标注
3. **易于扩展**：微服务架构，模块解耦
4. **成本可控**：开源技术为主
